# ABC-FP

Adapted from [ABC-Enhancer-Gene-Prediction](https://github.com/broadinstitute/ABC-Enhancer-Gene-Prediction) (Broad Institute).

This directory contains a modified version of the Activity-by-Contact (ABC) model, adapted to use TF footprinting data (FOODIE) as the chromatin accessibility signal for enhancer-gene linking predictions (ABC-FP-Max).
